Introduction

Todo List Pro est une application complète de gestion de tâches qui fonctionne dans votre navigateur. Elle vous permet d'organiser vos tâches avec de nombreuses fonctionnalités avancées, dont la synchronisation avec Google Agenda.

Démarrage

  1. Ouvrir l'application : Double-cliquez sur le fichier HTML ou ouvrez-le dans votre navigateur préféré.
  2. Sauvegarde des données : L'application fonctionne en local. Utilisez la fonction "Exporter" pour sauvegarder vos données et "Importer" pour les restaurer.
  3. Synchronisation cloud : Connectez l'application à Google Agenda pour sauvegarder vos tâches dans le cloud et y accéder sur différents appareils.

Synchronisation Google Agenda

Connexion à Google

  1. Cliquez sur le bouton "Google Agenda" dans la barre d'outils.
  2. Dans la fenêtre qui s'ouvre, cliquez sur "Se connecter avec Google".
  3. Suivez les instructions pour autoriser l'accès à votre compte Google.
  4. Une fois connecté, vous pourrez configurer les paramètres de synchronisation.

Synchronisation des tâches

  • Synchronisation manuelle : Cliquez sur "Synchroniser maintenant" dans les paramètres Google pour synchroniser immédiatement toutes les tâches.
  • Synchronisation automatique : Activez l'option dans les paramètres pour que les tâches se synchronisent automatiquement.
  • Tâches individuelles : Vous pouvez choisir quelles tâches synchroniser en cochant l'option lors de la création ou modification.
Astuce : Les tâches synchronisées avec Google Agenda sont marquées d'un indicateur "G" dans la liste des tâches et le calendrier.

Gestion des conflits

Lorsque des modifications sont effectuées à la fois localement et sur Google Agenda, l'application détecte les conflits et vous propose de les résoudre :

  • Choisissez entre la version locale et la version Google pour chaque tâche en conflit.
  • Utilisez les boutons "Tout garder en local" ou "Tout importer de Google" pour résoudre rapidement tous les conflits.

Fonctionnalités principales

➕ Ajouter des tâches

  • Ajout rapide : Entrez simplement le titre de votre tâche dans le champ principal et appuyez sur Entrée.
  • Ajout détaillé : Cliquez sur "Plus d'options" pour accéder au formulaire complet avec :
    • Catégorie
    • Date et heure d'échéance
    • Priorité (basse, normale, haute)
    • Temps estimé
    • Récurrence
    • Tags
    • Description détaillée
    • Synchronisation Google Agenda

Aide-mémoire des indicateurs visuels

Pour vous aider à identifier rapidement l'état et la priorité de vos tâches, l'application utilise plusieurs indicateurs visuels :

  • Bordure rouge Tâche en retard
  • Indicateur "G" Tâche synchronisée avec Google Agenda
